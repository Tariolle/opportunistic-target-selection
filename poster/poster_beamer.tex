\sechead{1}{Setup \& Motivation}
Score-based black-box attacks see probabilities, not gradients. Standard untargeted losses mainly lower the true-class confidence:
\[
\mathcal{L}_{\mathrm{untarg}}(x',y)=P(y\mid x')\quad\text{or}\quad \log P(y\mid x').
\]
\begin{itemize}
  \tightitems
  \item Probability mass is freed, but no non-true class is chosen.
  \item The trajectory can drift across several class basins.
  \item Query budget is wasted before misclassification.
\end{itemize}

\bluebox{\textbf{Goal.} Pick a viable target from the trajectory, then commit early to avoid class drift.}

\textbf{Gap.}
\begin{itemize}
  \tightitems
  \item Targeted attacks need a target before the run starts.
  \item CE/probability losses often do not track a competitor.
\end{itemize}

\sechead{2}{Method: Opportunistic Target Selection}
OTS keeps the proposal mechanism unchanged. Only the loss schedule changes after a short untargeted prefix.

\bluebox{%
\textbf{Algorithm.}
\begin{enumerate}
  \tightitems
  \item Run the original untargeted attack for $T$ iterations.
  \item Lock onto the leading non-true class
  \[
  t=\arg\max_{k\neq y}P(k\mid x'_T).
  \]
  \item Continue the same attack, but maximize the target confidence $P(t\mid x')$.
  \item Stop when the classifier prediction is no longer $y$.
\end{enumerate}}

\begin{itemize}
  \tightitems
  \item Irreversible lock: no repeated exploration.
  \item Stable for $T\in\{5,\ldots,500\}$: rankings appear early.
\end{itemize}

\textbf{Oracle target:} same-seed untargeted endpoint, used as a trajectory-specific ceiling.

\sechead{4}{When Should OTS Help?}
\begin{center}
\begin{tabular}{@{}p{0.21\linewidth}p{0.23\linewidth}p{0.42\linewidth}@{}}
\toprule
Attack & Direction source & Expected OTS effect \\
\midrule
SimBA & No target tracking & Large gain from early commitment \\
Square CE & $y$-only CE & Margin-like gain \\
Bandits & Gradient estimate & Small or negative gain \\
\bottomrule
\end{tabular}
\end{center}

\begin{itemize}
  \tightitems
  \item Helps when the base loss drifts.
  \item Can hurt when the base attack already has a direction.
\end{itemize}

\sechead{9}{Loss Interpretation}
\begin{center}
\begin{tabular}{@{}lcc@{}}
\toprule
Square Attack, ResNet-50 & Success & Mean iter. \\
\midrule
CE untargeted & 85.0\% & 2,601 \\

\begin{itemize}
  \tightitems
  \item CE drift is repaired by OTS.
  \item Margin loss already tracks the competitor, so OTS is redundant.
\end{itemize}

\sechead{10}{Limits \& Future Work}
\begin{itemize}
  \tightitems
  \item Robust ImageNet models: flat success across variants.
  \item No medium-difficulty regime where commitment pays off.
\end{itemize}

\textbf{Future work.}
\begin{itemize}
  \tightitems
  \item CIFAR-10, $L_2$ budgets, randomized defenses.
  \item Adaptive re-locking if the target drops in rank.
\end{itemize}

\sechead{11}{Takeaways \& Links}
\bluebox{%
\textbf{Take-away.}
\begin{itemize}
  \tightitems
  \item Explore briefly.
  \item Lock the strongest non-true class.
  \item Useful only when the base attack drifts.
\end{itemize}}
